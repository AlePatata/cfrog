# Competitive Frogramming

This is a project for organize and structure a folder with our codes for programming competitive problems. For now has the following features

- init: initialize the index file called cfrog.json in the root of the project
- show: shows the cfrog.json file
- add: creates a cpp file with the parameter name, then it adds to cfrog.json file

## Instalation

Using uv
```bash
uv add cfrog
```
